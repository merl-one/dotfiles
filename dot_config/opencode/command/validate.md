# Validate Plan Implementation

1. Read the provided implementation plan `$ARGUMENTS`.
2. Cross-reference the checked `[x]` checkboxes with actual committed/staged code. 
3. Verify test coverage for new functionality by invoking tests in terminal.
4. If everything passes, update frontmatter to status=completed, move the plan file from `plans/active/` to `plans/completed/`.
