# PR Creation

1. Determine if a PR exists. Analyze diffs via git terminal.
2. Read `.github/PULL_REQUEST_TEMPLATE.md` if it exists.
3. Draft a PR title and detailed description mapped to the template.
4. Execute validation checks (linting/tests).
5. Stop and present the drafted description for user approval before making the GitHub PR.
