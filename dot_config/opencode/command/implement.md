# Execute Implementation Plan

## Process

1. Read the implementation plan located at `$ARGUMENTS`.
2. Check existing progress by evaluating the `[ ]` and `[x]` checkboxes.
3. Begin the next uncompleted phase.
4. Implement the required code changes.
5. Use the terminal to run relevant commands or scripts to verify your implementation. Don't guess; run it.
6. Once the phase is 100% verified, edit the plan file to check off the item `[x]`. 
7. Pause and ask the human if you should proceed to the next phase. Do NOT automatically proceed.
